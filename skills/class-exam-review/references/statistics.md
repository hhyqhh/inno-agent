# 统计口径

所有数字必须由作答明细计算得出，可追溯。下面定义每个字段的精确含义。

## 班级总体

| 字段 | 计算 |
| --- | --- |
| `headcount` | 实考人数（剔除缺考）。 |
| `average` | 实考学生总分算术平均。 |
| `max` / `min` | 实考学生总分最大/最小。 |
| `median` | 总分中位数。 |
| `stddev` | 总分标准差（总体，n）。 |
| `scoreBands` | 分数段人数，默认按满分 100 时段：`[0,60) [60,70) [70,80) [80,90) [90,100]`。满分非 100 时按比例缩放分段并注明。 |
| `passRate` | 达到满分 60% 的人数占比（可由教师在配置中改阈值）。 |

## 逐题统计

### 客观题（single / multiple / judge）

- `accuracy`（正确率）= 作答正确人数 ÷ 该题实作答人数。空白/未作答不进分母。
- `distribution`（选项分布）= 每个选项（含多选的字母组合，已内部排序归一）的人数与占比。
- `topWrong`（高频错误）= 占比最高的非正确选项，附带人数、占比。
- 多选题部分给分时（如漏选得部分分），第一版仅按“完全正确”计正确率，并在结果中注明“部分给分未计入正确率”。

### 主观题（subjective）

- `average`（平均得分）= 全体实作答学生该题得分算术平均。
- `scoreRate`（得分率）= 平均得分 ÷ 满分。
- `scoreBuckets`（得分段人数）：按满分等分四段 `[0,25%) [25%,50%) [50%,75%) [75%,100%]`，统计每段人数与占比。
- `zeroCount`：零分人数及占比（零分率过高通常是方法性错误，单列）。

## 知识点统计

- 对每个知识点 `k`，聚合其下所有题目：`scoreRate_k = Σ(学生实得分) ÷ Σ(学生该题满分)`，按分值加权而非简单平均。
- 知识点题目数 < 2 时标注“样本少，仅供参考”。
- 输出 `knowledgeStats`：按得分率升序列出，标注薄弱（`scoreRate < 0.6`）、合格（`0.6–0.8`）、掌握（`> 0.8`）。

## 错误原因诊断

### 客观题

对每道 `accuracy < 0.7` 的客观题，结合 `topWrong` 与题干给出**可能原因**（措辞用“疑似/可能”）：

```
第8题 正确率：31%

主要错误：
1. 42% 选择 B —— 疑似混淆充分条件与必要条件
2. 18% 选择 D —— 疑似忽略题干“至少”
3.  9% 计算正确但选项对应错误
```

诊断依据优先级：① 题干关键词与高频错误项的语义关系；② 知识点历史易错（若有 L2 记忆）；③ 选项结构（干扰项设计）。无依据时只描述“X% 选择 Y”，不臆测原因。

## 主观题错误归因表

主观题按学生作答（如教师上传若干份典型卷）或得分段推断，归入下列类别。归因须给出来自作答的证据（引用作答片段或得分段特征），不得凭空归类。

| 归因 | 典型证据 |
| --- | --- |
| `formula_select` 公式/定理选择错误 | 用错公式；步骤中出现的核心关系式与本题知识点不符 |
| `missing_step` 漏写必要步骤 | 跳步导致得分点未覆盖；中间关键推导缺失 |
| `calculation` 计算错误 | 思路正确但中间数值算错；得分集中在“方法分”，结果分缺失 |
| `condition_incomplete` 条件使用不全 | 忽略定义域/隐含条件/边界；题干某条件未参与推导 |
| `expression` 表达不规范 | 方法正确但符号、单位、术语不规范；证明逻辑跳跃 |
| `prerequisite_gap` 前置知识缺失 | 在本题所需的前置步骤即出错，无法进入本题主干 |

> 第一版不自动识别复杂数学证明过程的全部细节，归因以“得分段特征 + 教师上传典型卷”为主，缺证据时标“待教师补充典型卷”。

## 分母与缺考

- 任一题的统计分母都是**该题实作答人数**，不是班级总人数；空白、缺考、无效填涂剔除。
- 全班缺考人数在概览单列，不参与任何百分比。
