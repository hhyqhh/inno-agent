# 个人订正单模板

为每名学生生成一份只属于自己的订正内容，聚焦其本人错题，不重复整班讲评。

## 模板

```
{学生编号}的个人订正单（{学科} · {考试标题}）

你的重点订正题：{题号列表}

主要问题：
- {问题类型1，如：条件遗漏}
- {问题类型2，如：函数性质理解不完整}
- {问题类型3，如：计算过程缺少检验}

建议完成：
- 原题订正 {n1} 题：{题号列表}
- 同类练习 {n2} 题：见补偿练习 {题号→对应原题}
- 知识点复习 {n3} 项：{知识点列表}

本次总分：{total}（班级均分 {classAvg}，你的位置：{band}）
```

## 规则

- **学生编号**：使用学号或脱敏编号（如 `S07`），订正单默认不写真实姓名；若教师需要打印分发，由教师在交付时勾选“显示姓名”。
- **重点订正题**：取该生答错且班级讲评价值较高的题（至少 2 题，至多 6 题），不堆砌全部错题。
- **主要问题**：从其错题中归纳 2–4 类，用 [主观题错误归因表](./statistics.md#主观题错误归因表) 或客观题诊断结论，措辞面向学生可行动（“下次先检查定义域”而非“你定义域不行”）。
- **建议任务量**：按学生本次总分段差异化——低分段侧重基础巩固，高分段侧重提高题；任务总量控制在 30–45 分钟可完成。
- **班级位置**：只给分段（如“70–80 段”），不公布排名。
- **数据隐私**：订正单只在工作区内生成，每生一份独立文件；不发送外部服务，不集中暴露他人成绩。
